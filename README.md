<h3>👍 프로젝트</h3>
울산KH정보교육원을 통하여 '독서기록 서비스'를 제공하는 팀프로젝트를 수행하고 그 중에서 회원CRUD를<br>
진행하였습니다.<br> 
SpringBoot Security 를 이용하여 DB 암호화 기능을 적용하였으며, Oracle 18c XE 에서 정규식을 이용한<br>
제약조건 등록과 프로시저 및 스케줄러 기본 기능 사용하였습니다.<br>
